# Overview

This project analyzes a digital marketing campaign dataset using **Microsoft Excel** for interactive dashboards and **Python (Pandas, Matplotlib, Seaborn)** for deep data insights.

We uncover:

- Key customer segments and spending habits
- Campaign performance based on offer type and communication channel
- Correlation between demographic features and conversion rates

# Tools Used

- Microsoft Excel (Power Query, Pivot Table, Dashboard)
- Python (Pandas, Seaborn, Matplotlib)
- Google Colab (for notebooks)
- GitHub (version control and hosting)
